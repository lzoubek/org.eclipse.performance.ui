Builds Comparison view

This view shows the scenarios status table of the difference between two builds selected in the Builds view.

The comparison between the two builds is shown for each component in a separated tab which contains a table made of all scenarios (lines) and all configuration (columns).
Note that this format is similar than the one used to generate performance results HTML pages...
